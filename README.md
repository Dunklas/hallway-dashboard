# Hallway Dashboard
App to show public transport and weather for a given coordinate.

## Prerequisites
 - musl-tools
   - For building lambda

## Secrets
  These secrets need to be set in Github repo.

   - AWS_ACCESS_KEY_ID
   - AWS_SECRET_ACCESS_KEY
   - S3_ARTIFACT_BUCKET
   - WEATHER_API_KEY
   - WEATHER_LONGITUDE
   - WEATHER_LATITUDE
   - PUBLIC_TRANSPORT_API_KEY
   - PUBLIC_TRANSPORT_STOP
   - PUBLIC_TRANSPORT_DIRECTION
     - Optional
